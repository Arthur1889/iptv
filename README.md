📺 高画质 IPTV 自动择优工具

> **基于物理探测的直播源净化方案** > 自动从全球/国内源抓取链接，通过 `ffprobe` 实测画质，为 Apple TV (APTV) 用户提供极致的观看体验。

---

🌟 项目亮点

* **⚡ 深度画质跑分**：不看标签看实效，实测 **物理分辨率** 与 **真实码率**。
* **🎯 极致去重择优**：同频道全网比对，自动保留质量最高（真 4K/1080P）的唯一线路。
* **🎨 APTV 深度适配**：内置 [fanmingming/live](https://github.com/fanmingming/live) 接口，自动关联高清台标与节目预告 (EPG)。
* **🔢 科学频道排序**：修正 CCTV 系列频道排序逻辑，符合国内用户收视习惯。
* **📈 过程透明可见**：配备 `tqdm` 可视化进度条，实时监控分析效率。
